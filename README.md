# Server-Express-PostgreSQL
 Testing server Express on the TypeScript with database PostgreSQL

# Technology stack
- Express.
- TypeScript (tsc).
- Dotenv.
- Nodemon.
- PG (Node-Postgres)

## Used to create a database [PostgreSQL](https://www.postgresql.org)
